This page is a work in progress as I continue to add projects from my Bootcamp experience. Please check back soon for additional projects!

Art Nouveau

Bootcamp project that creates a gallery exhibit (themed on our interests) that allows users to create, edit, and delete exhibits.

Technologies: Laravel and PHP provide the framework for the project. PostgreSQL was used to create the database. Boostrap, HTML, and CSS provide styling while Heroku is utilized to host the site.

BLumbler

Bootcamp project that creates a Tumbler-esque site that allows users to create, edit, and delete blog posts.

Technologies: Laravel and PHP provide the framework for the project. PostgreSQL was used to create the database. Boostrap, HTML, and CSS provide styling while Heroku is utilized to host the site.

Time Telling

Bootcamp project that creates a digital clock with alarm and timer functions.

Technologies: VueJS handles the time keeping logic while Bootstrap, HTML, and CSS provide basic styling.

Faux Restaurant Site

Bootcamp project creating a fake restaurant website.

Technologies: Bootstrap, HTML, and CSS provide basic styling and API utilization for the menu items.
